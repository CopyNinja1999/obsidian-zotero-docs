# 从 Zotero 导入批注

在本教程的这一部分中，我们将为您介绍将批注从 Zotero PDF 阅读器导入到 Obsidian 的过程。这个功能可以帮助你保持 PDF 阅读和笔记间的同步。

通过 Obsidian-zotero 工具，这个过程已经被简化为几个点击操作，省去了手动复制批注的麻烦。

import { Steps, Callout } from "nextra/components";

## 将批注导入到文献笔记中 [#import-annotations]

<Steps>

### 在 Zotero 库中选择文献 [#select-literatures]

打开您的 Zotero 文献库，并选择要导入批注的 PDF 文件。

<Callout>
您可以通过按住 `Ctrl` 键（或在 Mac 上按住 `Cmd` 键）并单击每个所需的 PDF，或者按住 `Shift` 键并选择多个 PDF。
</Callout>

### 导入 [#trigger-import]

1. 右键单击所需的 PDF。
2. 在弹出的菜单中，选择 "Obsidian Actions" > "Create Literature Notes"

![从 Zotero 创建文献笔记](/img/zotero-create-notes.png)

### 在 Obsidian 中获取文献笔记 [#literature-notes-imported]

就是这样！您的批注现在已导入到一个文献笔记中，并且该笔记会自动在 Obsidian 中打开。默认情况下，该笔记保存在 Obsidian 的 "LiteratureNotes" 文件夹中。

</Steps>

在我们教程的下一部分中，我们将更深入地介绍在 Obsidian 中自定义模板，例如更改文献笔记中包含的属性。
