# 插入引用

使用 obsidian-zotero 插件，您可以轻松地在笔记中使用命令或编辑器补全插入引用。

<Steps>

### 触发引用补全

在您的笔记中键入 `[@`，您将看到一系列可供选择的引用。

### 搜索引用

您可以继续键入来搜索引用。

目前您可以通过标题、作者和年份进行搜索。

### 插入引用

选择要插入的文献，然后按下 `Enter` 键插入它的引用。默认情况下，它将以 [Pandoc 引用格式](https://pandoc.org/MANUAL.html#extension-citations) (`[@citekey]`) 插入引用。

您可以在按下 `Enter` 键之前，在最后留下 `/` 来插入替代引用格式。例如，`[@your_query/]` 将以替代格式插入引用。

---

</Steps>

可以在插件设置中配置引用格式。请查看[前一节](/getting-started/basic-usage/template-config)以获取更多详情。

## 命令

您还可以使用 `Zotero Plugin: Insert Markdown Citation` 命令来插入引用。
步骤与编辑器建议类似，唯一的区别是您可以使用 `Shift`+`Enter` 键来插入替代格式的引用。

默认情况下，此命令未绑定键盘快捷键。
