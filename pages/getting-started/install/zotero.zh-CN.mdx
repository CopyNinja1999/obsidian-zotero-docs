# Zotero 插件配置

<ZoteroInfo />

## 安装 [#install]

<Steps>

### 下载 [#download]

从 GitHub Release 下载 `zotero-obsidian-note-*.xpi` 插件: <ReleaseLink />

<Callout type="info">
  如果您使用的是火狐浏览器，则应右键单击链接并选择 `另存为链接...`
</Callout>

<Callout>
  您也可以访问 [GitHub Release](https://github.com/PKM-er/obsidian-zotlit/releases) 页面，手动下载标签为 <ReleaseTag /> 的最新版本。
</Callout>

### 打开插件管理器 [#open-add-on-manager]

主菜单选择 `工具` > `附件组件`

### 安装插件 [#install-extension-from-file]

1. 点击右上角齿轮⚙图标，在菜单中选择 `Install Add-on From File...`
2. 选择刚才下载的 `.xpi` 文件, 单击 `Install`

<Callout>
  你也可以直接将 `.xpi` 文件拖放到插件管理器窗口中
</Callout>

### 重启 Zotero [#restart-zotero]

</Steps>

<Callout type="info">
  安装之后，插件会自动更新到新版本，因此理论上您只需执行一次此处描述的过程，如果插件未能自动更新，你可能需要重复此过程。
</Callout>
