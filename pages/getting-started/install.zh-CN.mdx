# 安装

为了确保所有功能都能正常工作，你需要安装以下两个插件：

1. 一个Obsidian插件`zotlit`，它提供了大部分功能，如插入引用、PDF注释查看等。
   - 安装Obsidian插件：<LatestBadge type="obsidian" href="/getting-started/install/obsidian" alt="Obsidian插件安装指南"/>
2. (可选) 一个Zotero插件`zotero-obsidian-note`，允许你在Zotero中打开链接的obsidian笔记，并从Zotero中导出项目到obsidian。
   - 安装Zotero插件：<LatestBadge type="zotero" href="/getting-started/install/obsidian" alt="Zotero插件安装指南"/>

此外，你还可以安装以下插件来增强你的使用体验：

- [Zotero Better BibTex](https://retorque.re/zotero-better-bibtex/)
   - 默认情况下，此插件会使用Better BibTex生成的citekey插入引文。我还没有测试其他的解决方案，但你可以在插件设置中更改引文格式。
- [Zotfile](https://www.zotfile.com/)
   - 你可以将PDF保存到Obsidian库中，并且该插件可以生成一个库内的wikilink，从而让你可以在obsidian内查看PDF文件。
