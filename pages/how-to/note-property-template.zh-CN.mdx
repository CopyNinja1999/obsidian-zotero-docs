# 笔记属性模板使用指南

请在下方评论区留下您的需求，我们将把它们添加到列表中。

## 只包括手动创建的标签

<Callout type="info">
这是在 v1.1.0 变更之前的默认行为。
</Callout>

要只包括手动创建的标签，而不包括从相关文献中提取的标签，请使用以下代码：

```ejs copy
tags: [<%= it.tags.filter(t => t.type === 0) %>]
```

<Callout>
在 Zotero 中，手动创建的标签的值为 0，而从相关文献中提取的标签的值为 1。
</Callout>

## 获取文献导入到 Zotero 的时间

```ejs copy
import-date: <%= it.dateAccessed %>
```

## 获取完整的作者列表

### 包括全名

```ejs copy
authors: [<%= it.authors %>]
```

### 包括名字或姓氏

```ejs copy
authors: [<%= it.authors.map(v => v.firstName) %>]
# 或者
authors: [<%= it.authors.map(v => v.lastName) %>]
```

<Callout>
请注意，`firstName` 和 `lastName` 属性的值可能为空。
</Callout>
