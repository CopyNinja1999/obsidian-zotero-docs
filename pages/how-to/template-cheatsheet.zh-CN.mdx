# 模板速查表

请在下方评论区留下您的经常用的模板，我们将把它们添加到列表中。

## 作者 [#authors]

### 作者缩写 [#authors-shorthand]

```ejs filename="Note Content" copy
[authors:: <%= it.authorsShort %>]
```

渲染结果类似 `[authors:: Gerasimavicius et al.]`。

## 摘要 [#abstract]

### 摘要输出为单行 [#abstract-oneline]

```ejs filename="Note Content" copy
## 摘要

<%= it.abstractNote.first().replace(/[\r\n]+/g, " ") %>
```

## 分组 [#grouping]

详见 [Object.groupBy](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Object/groupBy) 和 [Map.groupBy](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Map/groupBy)

### 按批注颜色分组 [#group-by-annotation-color]

```ejs filename="annotations" copy
<% const byColor = Object.groupBy(it, (annot) => annot.colorName); 
for (const color in byColor) { -%>

## <%= color %>
  <%_ for (const annot of byColor[color]) { %>

<%~ include("annotation", annot) %>
  <%_ } %>
<% } %>
```

### 按批注颜色分组（自定义颜色名称） [#group-by-annotation-color-customized-label]
  
```ejs filename="annotations" copy
<% const byColor = Object.groupBy(it, (annot) => annot.colorName);
const label = {
  "red": "红色",
  "orange": "橙色",
  "yellow": "黄色",
  "gray": "灰色",
  "green": "绿色",
  "cyan": "青色",
  "blue": "蓝色",
  "navy": "海军蓝",
  "purple": "紫色",
  "brown": "棕色",
  "magenta": "品红",
};
for (const color in byColor) { -%>

## <%= label[color] ?? color %>
  <%_ for (const annot of byColor[color]) { %>

<%~ include("annotation", annot) %>
  <%_ } %>
<% } %>
```
